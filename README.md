# Projeto Serliv

Projeto simples de exemplo criado para a aula (aula006). É um site estático com objetivo didático: demonstrar estrutura básica de páginas web, estilos com CSS e inclusão de imagens.

**Tecnologias usadas**

- **Linguagens**: HTML, CSS
- **Recursos**: imagens estáticas (pasta imagens)

**O que é este projeto**

O projeto `aula006` contém uma página estática (`index.html`) e estilos separados (`style.css`). Serve como exercício para aprender:

- estruturar um documento HTML;
- aplicar estilos com CSS (layout, cores, tipografia);
- incluir e referenciar imagens e recursos estáticos.

**Estrutura de arquivos**

- [index.html](index.html) — arquivo principal do site
- [style.css](style.css) — estilos do projeto
- imagens/ — pasta para imagens usadas no site
